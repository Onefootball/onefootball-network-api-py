    OneFootballNetwork¶

    ¶

    OneFootball Network API Client.

    __init__(self, login=None, password=None) special ¶

    Initialise client.

    Parameters:

    Name Type Description Default
    login Optional[str]

    email address you use to login on the OneFootball Network portal. If left empty, it is read from the LOGIN environment variable.

    		 None
    password Optional[str]

    password you use to login on the OneFootball Network portal If left empty, it is read from the PASSWORD environment variable.

    		 None

    Examples:


    # parse LOGIN and PASSWORD environment variables
of = OneFootballNetwork()
# or pass them directly
of = OneFootballNetwork(login="editor@football.com", password="mysecret")

    get_articles(self, external_id=None, feed_item_id=None) ¶

    List multiple posts created by you.

    When specifying external_id filter, this endpoint is expected to return a single entity most of the time. Bear in mind that two posts in two different languages can have the same external ID.

    Parameters:

    Name Type Description Default
    external_id Optional[str]

    The ID of the post as identified in an external system.

    		 None
    feed_item_id Optional[str]

    A comma separated list of the post feed item IDs. feed_item_id is neither the external_id or the onefootball_id.

    		 None

    Returns:

    Type Description
    PostsResponse

    the list of retrieved article objects.

    Exceptions:

    Type Description
    ValueError

    When the supplied filter combination is incorrect.

    Examples:


    of = OneFootballNetwork()
posts = of.get_articles(external_id="3")
# or
posts = of.get_articles(feed_item_id="990949,990967,995846")

    publish_article(self, article) ¶

    Publish an article to OneFootball.

    To use this endpoint, the creator account must have an integration of type "push" configured.

    Note

    You can use autocompletion on the NewPost object to see available attributes. The data model will take care of all data validation for you, so that you can focus on passing your content only.

    Parameters:

    Name Type Description Default
    article NewPost

    a NewPost object with the data of the article to publish

    		 required

    Returns:

    Type Description
    DetailedPost

    the published post

    Examples:


    from onefootball_network import OneFootballNetwork, NewPost

of = OneFootballNetwork()
article = NewPost(
    external_id="28961",
    source_url="https://www.clermontfoot.com/amical-orleans-clermont-annule/",
    language="fr",
    published="2020-08-10T08:28:58Z",
    title="Amical : Orléans – Clermont annulé",
    # fmt: off
    content='''<p>A la demande de l’US Orléans, le match amical du mercredi
        12 août opposant l’US Orléans et le Clermont Foot 63 à Moulins,
        est annulé.</p> <p>Le prochain match amical du CF63 est prévu le
        samedi 15 août contre Grenoble et sera retransmis sur le site de
        La Montagne dans le cadre de notre partenariat.</p>'''
    # fmt: on
)
post = of.publish_article(article)
print(post.onefootball_id)
