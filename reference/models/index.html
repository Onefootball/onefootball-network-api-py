    models¶

    PostUpdate¶

    ¶

    Article attributes that can be updated.

    content: HtmlBody pydantic-field required ¶

    The content of the article, which must be in correctly-formatted HTML. Please see this link for important details on acceptable HTML content.

    image_height: int pydantic-field ¶

    The image’s height in pixels.

    image_url: HttpUrl pydantic-field ¶

    An optional field for the URL of the article’s featured image.

    image_width: int pydantic-field ¶

    The image’s width in pixels.

    language: Language pydantic-field required ¶

    The language of the article.

    modified: datetime pydantic-field ¶

    The time that the article was last updated. If in doubt, use the current time.

    published: datetime pydantic-field required ¶

    The time that the article was published. If in doubt, use the current time.

    source_url: HttpUrl pydantic-field required ¶

    The link to the article as published on your website. Place your homepage url if the article is not published anywhere else.

    title: ConstrainedStrValue pydantic-field required ¶

    The title of the article.

    Config ¶

    Custom model config.

    __json_encoder__(obj) special staticmethod ¶

    partial(func, args, *keywords) - new function with partial application of the given arguments and keywords.

    is_provided_with_image_url(v, values) classmethod ¶

    If image_url is provided, image_width and image_height should be provided as well.

    override_default(v) classmethod ¶

    Handle misleading defaults sent by backend.

    Defaults: 'image_url': '', 'image_width': 0, 'image_height': 0

    NewPost¶

    ¶

    A new post payload.

    draft: bool pydantic-field ¶

    An optional boolean to be used for testing purposes. If set to True, the article will not be made visible to OneFootball users. If not povided, the article will by default be made available to OneFootball users.

    external_id: str pydantic-field required ¶

    The ID of the article as defined in your system. It must be unique (within a given language) within your own system.

    partial(func, args, *keywords) - new function with partial application of the given arguments and keywords.
